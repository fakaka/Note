# Vue-cli 3.x

## 使用 Vue-cli 2.0
使用vue官方的脚手架工具创建vue项目
---
需要环境nodejs和npm
---

### 1. 安装vue-cli  

    npm install vue-cli -g

### 2. 使用vue-cli创建项目

    vue init webpack <project-name>
    
### 3. 运行项目
    
先进入目录    

    cd <project-name>

再安装依赖

    npm install
    
运行开发环境

    npm run dev

构建项目

    npm run dev
