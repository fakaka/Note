# Vue-cli 3.x

## 使用 Vue-cli 2.0
> 使用 vue 官方的脚手架工具创建 vue 项目
::: tip
需要环境nodejs和npm
:::

### 1. 安装vue-cli  

`npm install vue-cli -g`

### 2. 使用vue-cli创建项目

`vue init webpack <project-name>`

### 3. 运行项目
    
1. 先进入目录 `cd <project-name>`

2. 再安装依赖 `npm install`
    
3. 运行开发环境 `npm run dev`

4. 构建项目 `npm run build`
